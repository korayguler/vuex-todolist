# vuex-todolist

## Project setup

```
# Clone this repository
git clone https://github.com/korayguler/vuex-todolist

# Go into the repository
cd vuex-todolist

# Install dependencies
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
